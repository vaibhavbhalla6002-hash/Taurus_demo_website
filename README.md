## 🌿 About the Project

**Taurus** is a multi-page, interactive zodiac personality website built as part of the *Hack The Horoscope* design hackathon under **Innerve 2026**. The project translates the Taurus zodiac archetype — *The Sensualist* — into a fully functional, emotionally intelligent digital experience using only **HTML, CSS, and Vanilla JavaScript**.

Every design decision — from color to animation speed — reflects Taurus personality traits: grounded, luxurious, sensual, patient, and nature-loving.

---

## ✨ Features

- 🌌 **Animated Hero Section** — Taurus constellation with floating nature particles
- 🧠 **Personality Trait Wheel** — Interactive radar chart that animates on load; click traits to expand descriptions
- 🎨 **Live Mood Board** — Theme shifts dynamically based on selected Taurus mood (Earthy / Luxe / Wild)
- 🌍 **Earth Elements Section** — Click-to-reveal cards for Venus, the Bull symbol, and the Emerald gemstone
- 💞 **Compatibility Zodiac Wheel** — Auto-rotating wheel; hover and click to reveal relationship dynamics
- ⏰ **Live Venus Hour Clock** — Real-time clock ticking in the navbar
- 🌟 **Easter Egg** — Click the constellation 3 times to trigger a gold particle burst
- 📱 **Fully Mobile Responsive**

## 🚀 Getting Started

No installations or dependencies required. This is a pure frontend project.

```bash
# Clone the repository
git clone https://github.com/vaibhavbhalla6002-hash/Taurus_demo_website.git

# Navigate to the project folder
cd Taurus_demo_website/taurus

# Open in browser
open index.html
```

Or simply drag and drop `index.html` into any browser.

## 🎓 Hackathon Context

This project was built for **Hack The Horoscope**, a personality-driven UI/UX hackathon under **Innerve 2026** at IGDTUW. The challenge was to use zodiac archetypes as design frameworks — not as astrology, but as a system for building emotionally intelligent digital experiences.

**Design philosophy followed:**
- Personality mapping → Design translation → Functional prototype
- Emotional tone → UI decisions → Interaction behavior
- Visual identity → Usability → Real-world product thinking
